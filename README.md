## Gray

A search engine with indexing, article retrieval (sub 1 second) and ranking. Implemented TF-IDF, cosine similarity and latent Dirichlet allocation for topic modelling.

- https://www.ardendertat.com/2011/05/30/how-to-implement-a-search-engine-part-1-create-index/ (Blog referenced)
- https://www.kaggle.com/datasets/ad5e75c9f4643f587ab945243c5b46381eb1698d628b7d1815b75010615c5c0f (Dataset used)
